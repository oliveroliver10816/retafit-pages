Set up our own Google Indexing API

~10 minutes, one time. Do it all in one Google account. The Indexing API is free — no credit card needed.

⚠ The one thing to save: whichever Google account (email) you use for step 1 — tell me that email at the end. That's the exact thing we lost last time. I'll record it permanently so this never happens again.
  1. Pick your account
    Log into the Google account (email) you want to own this — a fresh one is totally fine. Make sure it's the only one active, or note it, so you don't create things under the wrong account.
  2. Create a project
    Go to console.cloud.google.com/projectcreate → Name it retafit-indexing → Create → wait for it to finish, then make sure it's selected in the top bar.
  3. Turn on the Indexing API
    Go to console.cloud.google.com/apis/library/indexing.googleapis.com → confirm your new project is selected (top bar) → click Enable. (If it asks about billing, it shouldn't for this API — tell me if it does.)
  4. Create a service account
    Go to console.cloud.google.com/iam-admin/serviceaccounts → + Create service account → name it indexer → Create and continue → skip the "grant access" steps → Done.
  5. Download its key file
    Click the indexer account you just made → Keys tab → Add key → Create new key → JSON → Create. A .json file downloads. This is the secret — send me the whole contents (open it in Notepad/TextEdit, copy everything, paste to me). I store it locked-down, never on any site or repo.
  6. Make it an Owner in Search Console
    In the JSON you'll see "client_email": "indexer@retafit-indexing-…iam.gserviceaccount.com" — copy that robot email. Then GSC → tryretafit.com → Settings → Users and permissions → add that email as Owner (same as you did last time). Ping me if the Owner option is buried and I'll guide you.
  7. Tell me the account email
    Reply with (a) the JSON contents from step 5, and (b) the Google account email you used in step 1. I'll immediately push all 16 pages through the API — and permanently record the account so future-us never loses it.
